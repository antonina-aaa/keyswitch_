Мапи символів

Два std::map зберігають відповідності між клавішами двох розкладок.

🔍

Пошук по символу

Кожна буква шукається в мапі і замінюється відповідником з іншої мови.

🔠

Регістр не губиться

Велика буква залишається великою — isupper() + towupper().

🪟

Windows UTF-8

SetConsoleOutputCP(65001) вмикає кирилицю в терміналі.

main.cpp — ключовий фрагмент 
// Відповідність клавіш Англ → Укр
map<char, wchar_t> engToUkr = {
    {'q', L'й'}, {'w', L'ц'}, {'e', L'у'},
    {'r', L'к'}, {'t', L'е'}, // ... і так далі
};

static wstring toUkr(const string& text) {
    wstring result;
    for (char ch : text) {
        char lower = tolower(ch);
        if (engToUkr.count(lower)) {
            wchar_t ukr = engToUkr[lower];
            if (isupper(ch)) ukr = towupper(ukr);
            result += ukr;
        }
    }
    return result;
}

Цей код народився не з любові до автоматизації, а з хронічної втоми від передруковування тексту набраного не тією розкладкою.
